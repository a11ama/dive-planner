## 目的地資料格式規範

每份目的地 MD 文件**必須**按以下順序排列：

1. **⚡ 理想行程 × 費用**（**最頂端**，速查表）
   - 含多種時間方案（短/標準/長）
   - 每方案：天數、潛水+住宿費用、機票費用、每人總費用、需請假幾天
2. **✈️ 簡易行程表**（飛行時間、轉機次數，只寫到潛水店/登船，不排潛點）
3. 目的地特色
4. 海洋亮點
5. **🏪 潛店/船宿評比**（✅好評 + ❌差評，直接整合在評比卡內，**不另開報價表**）
6. 注意事項
7. 參考連結

## 請假難度標準（上班族視角）

我們都是上班族。台灣年假日曆：

- **0 天請假**：只用週末（週六-日），最多 2–3 天旅行
- **1–2 天請假**：接週末可打通 4–5 天，適合短程
- **3–5 天請假**：最長 9–10 天，適合中程（如配合春節/國慶）
- **6+ 天請假**：需刻意規劃，適合長程船宿（推薦春節/五一等長假）

**重要連假窗口：**

| 假期 | 日期 | 可打通天數（+請假2天）|
|------|------|------|
| 春節 | 2月初，約 7 天 | ~11 天 |
| 清明 | 4月初，約 4 天 | ~7 天 |
| 端午 | 6月初，約 3 天 | ~6 天 |
| 中秋+國慶 | 9–10月，視年份 | ~7–9 天 |
| 元旦 | 1月初 | ~4–5 天 |

> ⚠️ 在研究每個目的地時，**必須標注對應的假期窗口與請假天數**，而不是只給出任意日期。

## 費用估算原則（給 AI 看的）

> 費用範圍太寬等於沒有參考價值。請嚴格遵守：

1. **費用 = 總費用**：每人總費用 = 機票 + 潛水/船宿 + 住宿 + 必要交通（港口接送等）
2. **用旺季機票估算**：機票用目的地最佳潛水季的**旺季廉航參考價**，不用最低特價也不用商務艙
3. **縮窄區間**：每個方案的費用區間不超過 NT$1 萬，若無法縮窄請說明原因
4. **用標準行程天數**：通常以最推薦的天數（如 5D4N）來計算，不要把最短和最長行程放在同一個區間
5. **Operator 必須有官網連結**：每家業者資料必須包含官方網站連結（最好是含船艙/設施照片的頁面）

## 研究新目的地的流程（給 AI 看的）

請依照此流程處理每個新目的地請求：

1. **先查目錄表**：確認是否已有資料（`潛水目錄表.md`）
2. **研究費用與排期**：先找清楚費用再寫文件，確保頂部費用表準確
3. **找業者評價**：必須找到真實用戶反饋，含差評
4. **寫目的地 MD**：依格式規範（本 README 上方）
5. **更新目錄表**：新增一行到 `潛水目錄表.md`
6. **行程計畫**（如用戶指定出行時間）：呼叫子代理人做精確班機/船期搜索，輸出帶 Skyscanner 深連結的計畫
